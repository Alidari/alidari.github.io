A mobile project that detects trucks and other vehicles. When a truck arrives, the system detects the license plate area and tries to read the plate. Images are sent from the mobile client to a Flask server, which processes them with AI models. If the vehicle is not a truck or the plate is not registered, a warning is issued. If the truck's plate is registered, the mobile client sends a request to an ESP32, which lights up an LED to simulate opening a gate.
